# 快速开始

## 安装

使用 npm、yarn 或 pnpm 安装核心功能：

```bash
# npm
npm install @drag-list/core

# yarn
yarn add @drag-list/core

# pnpm
pnpm add @drag-list/core
```

如果你使用 Vue 3，可以安装 Vue 组件包：

```bash
# npm
npm install @drag-list/vue

# yarn
yarn add @drag-list/vue

# pnpm
pnpm add @drag-list/vue
```
